Attributes¶

SaveLayerRec.fBackdrop¶

If not null, this triggers the same initialization behavior as setting Canvas.SaveLayerFlags.kInitWithPrevious on fSaveLayerFlags: the current layer is copied into the new layer, rather than initializing the new layer with transparent-black.

This is then filtered by fBackdrop (respecting the current clip).
